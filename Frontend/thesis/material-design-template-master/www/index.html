About

This is an app about Cyber Diversity.  
The purpose of this project is to investigate the security issues that might occur in information systems, if multiple users own the same applications. Cyberdiversity as a term is based on biology and specifically in biodiversity. Cyberdiverse computer systems could prove to be more resistant to potential attacks than systems that tend to monoculture. Monocultures can be characterized as a population that consists of identical members that belong to the same type. In a monoculture, when a piece of malware manage to contaminate one member of the system, it can affect the rest of it due to the fat that they all share the same vulnerabilities.
